# Build Requirements

* A C++ compiler with C++14 support
* make
* At least Qt 5.10
* The following Qt modules:
	- Qt Core 
	- Qt Quick (qt5-declarative)
	- Qt QML
	- Qt Widgets
	- Qt Quick Controls 2 (qt5-quickcontrols2)
	- Qt Multimedia (qt5-multimedia)
* Aubio
